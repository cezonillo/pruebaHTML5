pruebaHTML5: Trabajo realizado para el curso de Extensión Universitaria "HTML5 y el futuro de la Web".
===========

Autor: Juan Ramón Martínez Cezón.
Descripción: En este trabajo se pretende resumir en forma de práctica todos los conocimientos que se han ido adquiriendo durante el curso.
La idea es realizar un apartado con un pequeño caso práctico, por cada uno de los puntos más destacados tratados durante el curso.
De este modo se ponen en práctica todas o casi todas las clases recibidas, de forma que el alumno consigue poner todos los conocimientos en práctica.

La Web está optimizada para google chrome, ya que es el único navegadores que soporta completamente todas las características HTML5 y CSS3 
utilizadas para el desarrollo de la práctica.

Se trata de un cubo con 9 enlaces, en los que al pinchar nos llevaría a un pequeño caso práctico de ejemplo.

1. Inserción de tipografías personalizadas (@font-face)
2. Vídeos en HTML5, y un par de enlaces que ejecutan acciones sobre el vídeo (Usando así la API HTML5)
3. Geolocalización. Un pequeño ejemplo, en el que al pulsar sobre el botón, obtenemos la geolocalización actual mediante la API HTML5.
4. Javascript. Se inserta un plugin jQuery que implementa una galería de imágenes.
5. Drag and Drop. Un ejemplo de la API Drag and Drop de HTML5 que permite arrastrar monedas a la hucha.
6. Utilización de Canvas y la librería jCanvas. Se trata de la implementación del popular juego de piedra, papel o tijera.
7. Sesiones. Un ejemplo de cómo actuan las sesiones sessionStorage y localStorage.
8. Web Semántica. A modo de teoría, se muestran los conocimientos básicos adquiridos sobre microdatos.
9. Diseño Responsivo. Un pequeño ejemplo del cada vez más utilizado diseño responsivo mediante css3 (@media)
